# API Testing Project

## Description
This project contains API testing for a REST API using Postman.
It includes different test scenarios to validate the API behavior.

## Tools Used
- Postman

## Project Structure
- Collection: contains all API requests
- Environment: contains variables used in requests

## Test Scenarios
- Positive test cases (valid inputs)
- Negative test cases (invalid inputs)
- Edge cases

## How to Run
1. Open Postman
2. Import the collection file
3. Import the environment file
4. Select the environment
5. Run the collection using Collection Runner

## Sample APIs Tested
- GET /users
- POST /login
- PUT /users/{id}
- DELETE /users/{id}
